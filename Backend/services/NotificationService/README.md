# Notification Service

Node.js + TypeScript gRPC service for managing user notifications (transaction alerts, account updates, security notifications).

## Scripts

- `npm run dev` — start with ts-node-dev
- `npm run build` — compile to dist
- `npm run start` — run compiled code
- `npm run proto:generate` — generate TS defs from `proto/notification.proto`

## Env

See `.env.example` for defaults.
